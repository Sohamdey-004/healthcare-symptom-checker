# Healthcare Symptom Checker

A simple rule-based Python application that analyzes user-reported symptoms and suggests possible health conditions with basic guidance. The project focuses on early awareness and accessibility and is not a medical diagnosis tool.

## Features
- Accepts multiple symptoms
- Provides possible condition and advice
- Simple and beginner-friendly
- Focused on healthcare awareness

## Technologies Used
- Python
- Command-line interface

## How to Run
1. Install Python
2. Run:
   py symptom_checker.py

## Disclaimer
This application is for educational purposes only and does not replace professional medical advice.
